HW1-2　隨機策略顯示與價值評估 Policy Evaluation

設定好起點、終點與障礙物後，點擊以下按鈕，系統將為每個格子 隨機生成行動策略（↑↓←→），並透過迭代策略評估計算每個狀態的價值 V(s)。

📐 MDP 設定：

  • 每步獎勵 r = -1，到達終點 r = 0（吸收狀態）
  • 撞牆或障礙物 → 原地不動，仍扣 1
  • 折扣因子 γ = 0.9，收斂門檻 θ = 1e-6

每格左上角顯示 座標 (r,c)，中間顯示 行動箭頭，下方顯示 V(s) 值。

HW1-3　價值迭代與最佳策略 Value Iteration

使用 Bellman 最優方程式反覆迭代，找出每個狀態的最佳價值 V*(s) 與最佳策略 π*(s)。 結果將覆蓋網格上的箭頭與數值，並以金色箭頭顯示最佳行動。

🔢 價值迭代演算法：

  • Bellman 最優更新：V*(s) = maxa [ R(s,a) + γ · V*(s') ]
  • 對所有狀態反覆同步更新，直到最大誤差 δ < θ = 1e-6
  • 從收斂的 V* 貪婪地提取最佳策略 π*(s) = argmaxa Q*(s,a)

最佳策略完成後，格子箭頭將變為 金色，代表在該狀態下的最佳行動。
